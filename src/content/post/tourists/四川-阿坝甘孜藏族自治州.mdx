人生总要去一趟远方吧！下一站行程，暂定西藏🚄。

我承认，是被强大的意念占领了，实在不想上班，没忍住。直接请假，从北京直奔四川的阿坝甘孜。

好了我承认上面我都是开玩笑的...
:::

起初我们考虑自驾。奈何我实在是个“小趴菜”，先不说阿坝地区的路况复杂，就是平时在市区开车我都费劲，所以自驾这个选项直接被我 PASS 了。

最终选择了最方便快捷的**飞机**✈️出行。您还别说，这交通工具有两把刷子（不好意思，我真是第一次坐飞机！）。其实坐飞机远没有大家想得那么复杂，基本流程如下：

1.  **购票**：不买票想吃霸王机？🤪
2.  **取票/值机**：如果需要报销凭证，可以到对应的航空公司柜台打印登机牌；个人出行可以直接使用电子登机牌进行登机操作。
3.  **安检**：取完票后，预留大约十分钟进行安检。安检完成后，正常进入候机楼找到飞机对应的登机口即可。

## 启程准备与行前攻略

一大早我就早早地赶到机场等着了。虽然第一次看到网上说要提前 2 小时到，但实际体验下来发现并没有那么夸张。

说说我这次为高海拔地区准备的物品吧：

### 行囊清单

| 分类 | 物品建议 | 备注 |
| :--- | :--- | :--- |
| **衣物** | 羽绒服 + 冲锋衣起步 | 必带围脖、围巾、手套、雪地靴等保暖物品。高海拔地区温差极大。 |
| **个人物品** | 换洗袜子、一套外衣裤 | 男孩子东西不多，简单就好。 |
| **摄影** | SONY A6000 + 两个镜头 | 我的“年迈”摄影搭档，记录沿途美景。 |
| **供氧** | 氧气瓶 | **务必准备充足！** 个人建议至少 **3 瓶**起步，团队建议团购。这是保命物资！ |
| **药物** | 肠胃药、感冒药、发烧药 | 高原反应可能引起肠胃不适或感冒症状，有备无患。 |
| **证件** | 身份证、护照、手机 | 这些东西老铁们应该忘不了吧！ |

> **关于红景天：** 有人说去高海拔地区要提前吃红景天，这个东西因人而异，建议听从医生建议或根据个人体质判断。

我们买的是早上 **7:15** 的机票，提前一个半小时到机场，时间还是比较充裕的。哎呀，激动得我是一夜都没睡！

我的行程是在淘宝上订的一个小团，价格是 **1340 元**，行程是两天，主要路线是：**毕棚沟 + 达古冰川**。具体行程下面介绍。

## 第一天：探索毕棚沟

我报的是一个 **2-6 人的精品小团**，基本上人都是满的。早上 **7:00 到 7:30** 左右，司机师傅会上门接人，然后出发前往毕棚沟景区。

第一天的早中晚饭都是自理，哥几个干脆在路上找了个饭店一起 AA 解决了一顿。

毕棚沟这个地方的游览流程略微复杂：司机把我们送到**半山腰**的旅客服务大厅，我们还需要在大厅外**等待大巴车**将所有人统一接送到景区内部！

![清澈见底的毕棚沟溪流与远处的雪山](https://img12.360buyimg.com/ddimg/jfs/t1/231439/4/4326/222432/656354a5Fc76b295c/35773bd99b5d0fc6.jpg)

![第一次在高海拔地区拍摄的雪山](https://img14.360buyimg.com/ddimg/jfs/t1/119447/37/40301/268457/656354a5F61385cda/dc5dfa71ae6b7293.jpg)

我非常喜欢这种在秘境中徒步的感觉。

:::warning[温馨提示]
高反提醒：在高海拔地区，一定要**缓慢行走**，切记不要奔跑或跳跃，因为部分人群很容易因此引发高原反应。
:::

![毕棚沟内景：松林、雪山、河流](https://img12.360buyimg.com/ddimg/jfs/t1/231281/38/4279/215926/656354b2F7425b90e/fc72f565e3923a0a.jpg)

玩了一天，吃吃喝喝后就准备休息了。

顺带提一嘴关于高反：我个人反应不太大，只是单纯感觉走路比平时喘得更快了。平时走十步喘一口，在这儿可能走五步就需要喘一口。另外，我带的氧气瓶在毕棚沟还没有用上。再次提醒大家：去高海拔地区，务必**量力而行**。

## 第二天：登顶达古冰川

### 夜宿海拔 3000m+

后续行程的住宿地在海拔**三千米以上**。我的天，那叫一个真冷！我们进了酒店，赶紧把所有能取暖的东西全部打开。就这样，我们度过了寒冷的一夜。

第二天一早，吃过早饭后，我们从 `羊茸哈德` 前往黑水县，大概半个小时左右的路程。达古冰川就在黑水县境内。

### 登顶：海拔 5000m

达古冰川的最高海拔大约 **5000 米**。冬天其实比较适合一些攀冰等极限运动（**普通人请别作死啊啊啊**）。

这次不是坐大巴上山，而是直接在半山腰坐**缆车**上山。我心想，这要是让我爬上去，得爬到什么时候啊！哈哈哈。

我们去的时候人特别多，缆车排队时间挺长的，大概在 **1 到 2 个小时**之间。而且从山下到山顶，坐缆车还需要 **15-20 分钟**左右。

:::important[🎥 知识点]
电影 **《中国机长》** 中，飞跃雪山🏔的场景就是在达古冰川这里取景拍摄的。
:::

## ✈️ 归途与小结

我们在当天晚上就连夜回到了成都，找个酒店住了一晚，第二天一早赶回了北京。

这一次旅行对我来说挺震撼的。这是我第一次去这么远、这么放松的地方。很难定义我真正拥有过什么，于是昏沉当中，这一年又要过去啦。

下一次的旅途，打算安排为期 7 天的**西藏**之旅，坐上绿皮火车。希望尽快提上日程吧。
